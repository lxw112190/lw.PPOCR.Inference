# lw.PPOCR.Inference v0.3.0

`v0.3.0` freezes the model manifest schema at version 1 and adds the first
correctness golden test. API version 1 and the four-backend architecture remain
unchanged.

## Highlights

- **Model manifest schema v1 frozen.** The JSON Schema now carries a stable
  `$id`, human-readable `description` fields on every property, and a formal
  commitment to backward-compatible additions only. See
  [model-manifest.md](../model-manifest.md) for the full specification.
- **Golden correctness test suite.** A new `lw.ppocr.golden` CTest runs
  OpenCV DNN on a reference image and compares the recognized text, bounding-box
  coordinates, and confidence scores against a recorded golden file. It catches
  OCR regressions that were previously invisible to region-count-only tests.
- **Golden recording mode.** Setting `LW_PPOCR_RECORD_GOLDEN=1` before the test
  regenerates the golden data file from the current build, making it
  straightforward to update baselines after an intentional model or parameter
  change.

- **模型清单 Schema v1 冻结。** JSON Schema 现携带稳定的 `$id`、每个字段的
  人类可读 `description`，并正式承诺仅做向后兼容的增量扩展。
- **Golden 正确性测试集。** 新增 `lw.ppocr.golden` ctest，在参考图片上运行
  OpenCV DNN，将识别的文字、包围框坐标和置信度与 golden 记录逐区域比对。
- **Golden 录制模式。** 设置 `LW_PPOCR_RECORD_GOLDEN=1` 即可从当前构建重新
  生成 golden 数据文件。

## Compatibility

- `LW_PPOCR_API_VERSION` remains `1`.
- No public struct layout, exported function, or ABI fingerprint changed.
- Model manifest schema v1 fields are frozen; existing manifests remain valid.
- Existing v0.2.0 callers can upgrade without source changes.

## Verification

The release build passed the ABI smoke test, ABI stability test, golden
correctness test, and real-model integration/stability tests for all four
production Runtimes on Windows x64.
