# lw.PPOCR.Inference v0.2.0

`v0.2.0` focuses on lifecycle, concurrency, and long-running stability while
keeping API version 1 and the existing four-backend architecture compatible.

## Highlights / 主要更新

- Added a public ABI stability gate covering 1,000 lifecycle cycles, 10,000
  sequential calls, 8-thread shared-handle calls, and multiple engine instances.
- Added real-model stress tests for OpenCV DNN, DirectML, OpenVINO, and TensorRT:
  100 sequential calls plus 4-thread concurrent calls per Runtime.
- Added private-memory and process-handle growth regression checks on Windows.
- Documented the engine threading and destruction contract in the public header.
- Unified native DLL and .NET assembly versions at `0.2.0`.

- 新增公共 ABI 稳定性测试：覆盖 1,000 次创建/销毁、10,000 次连续调用、
  8 线程共享实例调用和多实例并行。
- 四个真实后端均加入模型压力测试：每个 Runtime 执行 100 次连续识别和
  4 线程并发识别。
- 在 Windows 上检查私有内存与进程句柄增长，及时发现资源回收回归。
- 在公共头文件中明确线程安全、结果释放和销毁规则。
- 原生 DLL 与 .NET 程序集版本统一更新为 `0.2.0`。

## Compatibility / 兼容性

- `LW_PPOCR_API_VERSION` remains `1`.
- No public struct layout, exported function, or model manifest schema changed.
- Existing v0.1.0 callers can upgrade without source changes.

## Verification / 验证

The release build passed the ABI smoke test, ABI stability test, and real-model
integration/stability tests for all four production Runtimes on Windows x64.

本版本在 Windows x64 上通过 ABI 冒烟测试、ABI 稳定性测试，以及四个真实
Runtime 的模型集成与稳定性测试。
