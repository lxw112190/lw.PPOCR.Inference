# lw.PPOCR.Inference v0.8.0

`v0.8.0` publishes the ABI freeze candidate, compatibility matrix, and license
audit.  It is the last feature release before the v1.0.0 freeze.

## Highlights

- **ABI freeze candidate.**  All public C ABI symbols, the private Runtime
  contract, and the model manifest schema v1 are proposed for permanent
  freezing in v1.0.0.  See [abi-freeze-candidate.md](../abi-freeze-candidate.md)
  for the complete list.
- **Compatibility matrix.**  A new document records every verified OS, GPU,
  driver, and SDK combination across all four backends, plus known limitations
  and unsupported configurations.  See
  [compatibility-matrix.md](../compatibility-matrix.md).
- **License audit.**  Every direct and indirect dependency has been reviewed
  against the MIT license.  NVIDIA EULA compliance for TensorRT and CUDA
  redistribution has been assessed with concrete recommendations.
  See [license-audit.md](../license-audit.md).
- **C and Python examples.**  Added in v0.5.0, now stable.
- **Backend split packaging.**  `-Split` and `-Runtime` flags in the packaging
  script.
- **CI.**  GitHub Actions with VS preset build, ABI smoke/stability, and
  OpenCV DNN integration + golden test.

## Compatibility

- `LW_PPOCR_API_VERSION` remains `1`.
- No public struct layout or exported function changed.
- The ABI freeze candidate proposes no breaking changes — only formal
  documentation of the already-stable interface.

> `v0.8.0` is the final feature release.  From v0.9.0-rc.1 onward only bug
> fixes and documentation updates will be accepted.
